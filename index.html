🚀 Schnellstart-Anleitung

Um die Erweiterung kostenlos und ohne Einschränkungen zu nutzen, verwendet dieser Ticker die "Bring Your Own Key"-Strategie. Das bedeutet, du nutzt deinen eigenen Gratis-Schlüssel der weltweit bekannten Datenplattform football-data.org.

  1. Installiere die Erweiterung aus dem Chrome Web Store.
  2. Besuche die Website football-data.org und klicke oben rechts auf "Get a free API token".
  3. Trage deine E-Mail-Adresse ein. Du erhältst deinen persönlichen API-Schlüssel sofort per Mail.
  4. Öffne das Popup der Erweiterung in deinem Browser, klicke unten auf das Zahnrad (Einstellungen), füge deinen Schlüssel ein und klicke auf Speichern. Fertig!

🚀 Quick Start Guide

To use this extension completely free and without limits, it uses the "Bring Your Own Key" strategy. This means you will use your own free API key from the popular data platform football-data.org.

  1. Install the extension from the Chrome Web Store.
  2. Go to football-data.org and click "Get a free API token" in the top right corner.
  3. Enter your email address. You will receive your personal API token via email instantly.
  4. Open the extension popup in your browser, click the Gear Icon (Settings) at the bottom, paste your key, and click Save. All set!
