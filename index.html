🚀 Schnellstart-Anleitung

Um die Erweiterung kostenlos und ohne Einschränkungen zu nutzen, verwendet dieser Ticker die "Bring Your Own Key"-Strategie. Das bedeutet, du nutzt deinen eigenen Gratis-Schlüssel der weltweit bekannten Datenplattform football-data.org.

  1. Installiere die Erweiterung aus dem Chrome Web Store.
  2. Besuche die Website football-data.org und klicke oben rechts auf "Get a free API token".
  3. Trage deine E-Mail-Adresse ein. Du erhältst deinen persönlichen API-Schlüssel sofort per Mail.
  4. Öffne das Popup der Erweiterung in deinem Browser, klicke unten auf das Zahnrad (Einstellungen), füge deinen Schlüssel ein und klicke auf Speichern. Fertig!

🏆 Free vs. Pro Version

Du kannst den Ticker dauerhaft kostenlos nutzen. Wenn du das Projekt unterstützen und das volle Erlebnis freischalten möchtest, kannst du direkt in der App via ExtensionPay auf die Pro-Version upgraden.

🔒 Datenschutzerklärung (Privacy Policy)

Der Schutz deiner Daten hat bei uns oberste Priorität. Diese Erweiterung wurde so entwickelt, dass sie maximale Privatsphäre garantiert:

  • Keine Datenerfassung: Wir sammeln, speichern oder übertragen keinerlei persönliche Daten oder dein Surf-Verhalten.
  • Lokale Speicherung: Dein eingegebener API-Schlüssel sowie deine ausgewählten Spiele werden ausschließlich lokal auf deinem eigenen Computer im sicheren chrome.storage.local deines Browsers verschlüsselt gespeichert.
  • Keine eigenen Server: Es gibt keine externen Server unsererseits. Die Erweiterung kommuniziert direkt und exklusiv von deinem Browser aus mit der offiziellen API von football-data.org.
  • Sichere Zahlung: Die Pro-Freischaltung wird über den zertifizierten Drittanbieter ExtensionPay abgewickelt, welcher auf der sicheren Banking-Infrastruktur von Stripe basiert. Wir haben zu keinem Zeitpunkt Zugriff auf deine Zahlungsdaten.
